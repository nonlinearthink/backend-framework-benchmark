# Install Go

Go 1.22+ is required for `apps/fiber`.

## Windows

Download the installer from [go.dev/dl](https://go.dev/dl/) and run it, or:

```powershell
winget install GoLang.Go
```

Verify:

```powershell
go version
```

## macOS

```bash
brew install go
```

## Linux

See [go.dev/doc/install](https://go.dev/doc/install) for your distribution, or:

```bash
# Example (Ubuntu)
sudo apt install golang-go
```
